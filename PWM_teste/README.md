#### PWM e ADC10 ####

CPU: MSP430G2553 [(MSP-EXP430G2 LaunchPad)](http://www.ti.com/ww/en/launchpad/launchpads-msp430-msp-exp430g2.html)

Potenciômetro controlando a luminosidade de um LED

**Potenciômetro:**
  * Pin 1: VCC
  * Pin 2: P1.5
  * Pin 3: GND

**ADC10MEM:** GND = 0 --- VCC = 1023

**LED:** P1.6 (led verde)
